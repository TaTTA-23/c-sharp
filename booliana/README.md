# Booliana

Este programa demonstra o uso de expressões booleanas em C#.

## Descrição

O programa verifica se uma string contém determinadas substrings usando o método `Contains()`, que retorna um valor booleano (`true` ou `false`).

## Funcionalidades

- Demonstração de expressões booleanas
- Verificação de substrings em strings
- Uso do método `Contains()`

## Como executar

```bash
dotnet run
```

## Exemplo de saída

```
True
What does the fox say?
```
