# Loops e Iteração

## Descrição
Programa demonstrativo que explora diferentes tipos de **loops e iteração** em C#, cobrindo for, while, do-while e foreach para diversos cenários de repetição.

## Funcionalidades
- Demonstra loop for para contagem controlada
- Mostra loop while para repetição baseada em condição
- Explica loop do-while para execução mínima garantida
- Compara foreach para iteração em coleções
- Exemplifica cenários práticos de uso

## Conceitos Aprendidos
- **Loop for**: Iteração com contador controlado
- **Loop while**: Repetição baseada em condição
- **Loop do-while**: Execução garantida pelo menos uma vez
- **Loop foreach**: Iteração simplificada em coleções
- **Condições de parada**: Controlando quando parar
- **Incremento/decremento**: Modificando variáveis de controle

## Como Executar
```bash
cd loops-iteration
dotnet run
```

## Comparação dos Loops
| Loop | Quando Usar | Características |
|------|--------------|------------------|
| **for** | Contagem conhecida | Inicialização, condição, incremento |
| **while** | Condição dinâmica | Verifica antes de executar |
| **do-while** | Execução mínima | Executa pelo menos uma vez |
| **foreach** | Coleções | Mais seguro e legível |

## Casos de Uso
### Loop For
- Processamento de arrays por índice
- Contagem regressiva ou progressiva
- Iterações com padrão conhecido

### Loop While
- Leitura de arquivo até EOF
- Aguardando entrada do usuário
- Processamento até condição externa

### Loop Do-While
- Menus interativos
- Validação de entrada
- Tarefas que devem executar ao menos uma vez

### Loop Foreach
- Processamento de listas
- Busca em coleções
- Relatórios de dados
