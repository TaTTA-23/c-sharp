# Avaliação de Expressões Booleanas

Este programa em C# demonstra o uso de expressões booleanas em C#. Ele inclui exemplos de comparações entre strings e números, operadores de comparação como `!=`, `>`, `<`, `>=`, `<=`, e métodos de string como `Contains`, `Trim` e `ToLower`.

## Funcionalidades Demonstradas

- **Comparações de Igualdade e Desigualdade**: Verifica se valores são iguais ou diferentes.
- **Operadores de Comparação**: Avalia se um valor é maior, menor, maior ou igual, ou menor ou igual a outro.
- **Métodos de String**: Usa métodos como `Contains` para verificar se uma substring está presente, e `Trim` com `ToLower` para comparações insensíveis a maiúsculas/minúsculas.
- **Negação Lógica**: Aplica o operador `!` para inverter o resultado de uma expressão booleana.

## Código Principal

O código executa as seguintes avaliações:

- Compara dois inteiros (`a != b`).
- Compara duas strings (`s1 != s2`).

## Saídas Esperadas

- `True` para `a != b` (7 != 6).
- `False` para `s1 != s2` ("Hello" != "Hello").

O programa serve como um exemplo educacional para entender como as expressões booleanas funcionam em C#.
