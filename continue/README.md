# Continue Statement

Este programa demonstra o uso da instrução `continue` em loops C#.

## Descrição

O programa gera números aleatórios e usa a instrução `continue` para pular a execução quando determinada condição é atendida, demonstrando como controlar o fluxo de execução dentro de loops.

## Conceito

A instrução `continue` em C#:
- Passa imediatamente para a próxima iteração do loop
- Útil para filtrar valores sem sair completamente do loop
- Comumente usado com condições booleanas

## Funcionalidades

- Geração de números aleatórios
- Demonstração do do-while loop
- Uso da instrução continue para controle de fluxo

## Como executar

```bash
dotnet run
```

## Exemplo de saída

O programa gera números de 1 a 10, mas pula (não exibe) números >= 8, continuando até encontrar o número 7.
