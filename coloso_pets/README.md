# Coloso Pets

Aplicativo de gerenciamento de animais de estimação para o Contoso PetFriends.

## Descrição

Este programa é um sistema de menu interativo para gerenciar informações sobre animais de estimação. Ele utiliza arrays multidimensionais para armazenar dados e switch statements para navegação.

## Funcionalidades

- Listar todos os animais cadastrados
- Adicionar novos animais (em desenvolvimento)
- Editar informações de animais
- Visualizar animais por características específicas

## Dados armazenados

Para cada animal, o sistema armazena:
- ID
- Espécie
- Idade
- Descrição física
- Descrição de personalidade
- Apelido

## Como executar

```bash
dotnet run
```

## Menu de opções

1. Listar informações de todos os pets
2. Adicionar novo animal
3. Completar idades e descrições físicas
4. Completar apelidos e descrições de personalidade
5. Editar idade do animal
6. Editar descrição de personalidade
7. Exibir gatos com característica especificada
8. Exibir cães com característica especificada

Digite 'sair' para encerrar o programa.
